# Backend - Sistema de Sorteos (FastAPI)

Este directorio contiene el backend desarrollado con FastAPI para gestionar la lógica y datos del sistema de sorteos.

## Requisitos

- Python 3.8 o superior ([python.org](https://www.python.org))
- MySQL Server (puede ser XAMPP para desarrollo local)
- Paquetes Python:
  - fastapi
  - uvicorn
  - mysql-connector-python
